# Getting Started

This is an example of how you may give instructions on setting up your project locally.

# Prerequisites

 npm
  ```sh
  npm install npm@latest -g
  ```

# Installation

1. Clone the repo
   ```sh
   git clone https://github.com/E-comerce-BE-FE.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

# Usage

```sh
npm run dev
```
